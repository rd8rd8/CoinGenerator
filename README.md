# CoinGenerator
This generator oversamples data extracted from XRF analysis of coins.

### REPLICATION ###

To download CoinGenerator in a Linux system, please run:
<pre>
git clone https://github.com/rd8rd8/CoinGenerator.git
cd CoinGenerator
chmod +x *.sh
</pre>
